# Project Overview

This project is part of the Digital Literacy course, where I worked as a Student Digital Ambassador to create awareness about effective and safe use of digital tools.
The portfolio includes practical tasks such as designing an infographic, building a digital presence, exploring coding platforms, understanding professional communication, and studying cybercrime awareness.
The goal of this project is to develop essential digital skills required for academic growth and responsible online behavior.


# Module / Task Summaries

Task 1 – Digital Literacy Infographic
Created a visual infographic using Canva to explain digital literacy, its importance, useful tools, and safe internet practices in a simple and engaging way. Exported as PNG and saved in task 1 presentation

Task 2 – Student Digital Portfolio
Developed professional profiles on GitHub and LinkedIn to build an online presence and prepare for future academic and career opportunities.
GitHub Profile: https://github.com/paridhi1607

Task 3 – Coding & Collaboration Platforms

Practiced basic coding on HackerRank and created a Digital Literacy Awareness Quiz using Google Forms to understand online learning and collaboration tools.

Task 4 – Email Etiquette & Communication
Drafted professional emails and created a social media Do’s and Don’ts checklist to understand proper digital communication and responsible online behavior.

Task 5 – Cybercrime Awareness
Researched a cybercrime case study (UPI fraud/phishing) and created a prevention checklist to spread awareness about online safety.
